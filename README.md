# Employee Management System

A professional Employee Management System developed during my internship at **50 Hertz, Delhi**.

The application is built using **Python Flask**, **MySQL**, **HTML**, **CSS**, and **Chart.js**. It provides a complete employee management solution including authentication, employee records, department management, salary management, dashboard analytics, search functionality, and graphical reports.

---

# Project Overview

This project was developed to automate employee management processes inside an organization.

Instead of maintaining employee information manually in spreadsheets, the system stores everything securely inside a MySQL database and provides an easy-to-use dashboard for administrators.
